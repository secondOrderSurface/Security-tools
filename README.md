# Security-tools
My securety tools <br>
To install requirements execute:
```
pip install -r requirements.txt
```
SYN - а SYN scanner written in Python. It's considered faster and leaves fewer traces. <br>
XMas - XMas scanning is an unusual port scanning mode that allows you to bypass host filtering. A simple detector for this type of activity is written here. It's assumed that the three flags used in XMas scanning are not likely to appear on their own, so any packet with these characteristics triggers an alert.
